# 黄松松フィードバック修正ノート

## 問題1: 冒頭2-3秒に字幕がない
根本原因: 1599行目で `hook_end_time if cap['start'] < hook_end_time else cap['start']`
→ フックテキスト表示中（0-3秒）の字幕を全てスキップしている
→ 修正: フック表示中でも字幕は表示する（フックと字幕は位置が異なるので重ならない）
   フックは center_top、字幕は下部に表示されるので共存可能

## 問題2: フックが動画内容と無関係
根本原因: GPTプロンプトは既に「動画内容に関連」を指示しているが、
GPTが従わない場合がある。特にproduct_nameが空の場合。
修正: プロンプトをさらに強化 + バリデーション追加

## 問題3: カット不足
根本原因: silence_threshold_dbのデフォルト-30.0が緩すぎる
+ enable_content_cutのデフォルトがTrueだが閾値が緩い
修正: デフォルト値をより積極的に変更

## 問題4: PiP動画アップロード
根本原因: フロントエンドのaccept="image/*"が画像のみ
+ バックエンドのupload-product-imageが画像専用
修正: 動画アップロード用のUI+APIを追加

## 問題5: PiP短動画で3秒だけ表示
根本原因: _apply_pip_video_overlay関数が動画全体にPiPを表示
修正: 短い動画（5-10秒）では中盤の3秒だけPiP表示するロジック追加
