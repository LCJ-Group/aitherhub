AitherHub ウィジェットとは？

あなたのECサイトにワンタグで埋め込める、TikTok風のショッパブル動画フィードです。サイト訪問者は動画を見ながらそのまま商品を購入できます。

1

あなた専用のタグを確認

以下があなたのサイト専用のウィジェットタグです。このタグ1つで全ての機能が動作します。

<!-- AitherHub Widget -->
<script src="https://www.aitherhub.com/widget/loader.js"
data-client-id="cab30223" async></script>
このタグをコピーしておいてください。Step 2 で使用します。
2

Google Tag Manager（GTM）でサイトに追加

GTMを使えば、サイトのコードを直接編集せずにウィジェットを追加・管理できます。エンジニア不要で、いつでもON/OFFが可能です。

2-A. GTMアカウントを作成する （既にGTMをお持ちの方は 2-Bへスキップ →）

  1. tagmanager.google.com にアクセスし、Googleアカウントでログイン
  2. 「アカウントを作成」ボタンをクリック
  3. 以下の情報を入力： アカウント名：会社名またはブランド名（例：KYOGOKU Professional）
    国：日本
  4. コンテナの設定を入力： コンテナ名：サイトのドメイン（例：kyogokupro.com）
    ターゲットプラットフォーム：「ウェブ」を選択
  5. 「作成」をクリック → 利用規約に同意
GTMアカウント作成画面（参考）

2-B. GTMのコードをサイトに設置する （既に設置済みの方は 2-Cへスキップ →）

アカウント作成後、GTMから2つのコードスニペットが表示されます。これをサイトに1回だけ設置します。

  1. GTMが表示する1つ目のコード（<script>...</script>）を、サイトの <head> タグ内のなるべく上部に貼り付け
  2. GTMが表示する2つ目のコード（<noscript>...</noscript>）を、<body> タグの直後に貼り付け
  3. ファイルを保存してサイトをデプロイ（公開）
GTMスニペットの設置場所（参考）
この作業はサイトのHTMLを1回だけ編集する必要があります。EC-CUBEの場合は default_frame.twig、Shopifyの場合は theme.liquid に追加します。不明な場合はサポートにお問い合わせください。

2-C. GTMでAitherHubウィジェットタグを追加する

ここからが本題です。GTMの管理画面からウィジェットタグを追加します。

  1. GTM管理画面を開き、対象のコンテナを選択
  2. 左メニューの「タグ」をクリック → 右上の「新規」をクリック
  3. タグに名前を付ける（例：AitherHub Widget）
  4. 「タグの設定」をクリック → 「カスタム HTML」を選択
  5. Step 1 でコピーしたタグをそのまま貼り付け （下の「HTMLを入力」欄に貼り付けてください）
  6. 「トリガー」をクリック → 「All Pages」を選択
  7. 右上の「保存」をクリック
  8. GTM画面右上の「公開」ボタンをクリック → バージョン名を入力して「公開」
GTMでカスタムHTMLタグを追加する画面（参考）
これで完了です！ サイトにアクセスすると、右下にウィジェットのバブルが表示されます。
3

動作確認

タグを公開したら、サイトにアクセスして以下を確認してください。

  1. ページ右下に丸いボタン（バブル）が表示される
  2. バブルをクリックするとフルスクリーンの動画フィードが開く
  3. 動画が自動再生され、上下スワイプで次の動画に切り替わる
  4. 「購入する」ボタンが表示され、クリックで商品ページに遷移する
GTMの「プレビューモード」を使えば、公開前にテストすることもできます。GTM画面右上の「プレビュー」ボタンから利用できます。
GTMを使わない場合

代替方法：HTML直接埋め込み

GTMを使わずに、サイトのHTMLに直接タグを追加する方法です。

  1. サイトの共通テンプレート（EC-CUBEの場合：default_frame.twig）を開く
  2. </body> タグの直前に、Step 1 のタグを貼り付け
  3. ファイルを保存してデプロイ
<!-- サイトのHTML -->
<body>
... サイトのコンテンツ ...

<!-- AitherHub Widget（</body>の直前に追加） -->
<script src="https://www.aitherhub.com/widget/loader.js"
data-client-id="cab30223" async></script>
</body>

よくある質問

サイトの表示速度に影響はありますか？
ありません。ウィジェットは async 属性で非同期読み込みされるため、サイトの表示を一切ブロックしません。
動画の追加・変更はどうすればいいですか？
AitherHub管理画面からクリップを追加・削除できます。変更は即座にウィジェットに反映されます。サイト側の作業は不要です。
ウィジェットのデザインをカスタマイズできますか？
テーマカラー、CTAボタンの文言、ブランド名はAitherHub側で設定可能です。お客様のブランドに合わせた表示になります。
ウィジェットを一時的に非表示にしたい場合は？
GTM経由の場合はタグを「一時停止」にするだけです。HTML直接の場合はタグを削除またはコメントアウトしてください。
GTMとHTML直接、どちらがおすすめですか？
GTMを強く推奨します。GTMなら管理画面からいつでもタグのON/OFF、更新ができ、サイトのコードを触る必要がありません。複数のタグ（GA4、Facebook Pixel等）も一元管理できます。
複数のページにウィジェットを表示できますか？
はい。GTMのトリガーを「All Pages」にすれば全ページに表示されます。特定のページだけに表示したい場合は、GTMのトリガー条件でURLを指定できます。

サポートが必要ですか？

セットアップで困ったことがあれば、お気軽にお問い合わせください。
