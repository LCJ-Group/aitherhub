AitherHub ウィジェットとは？

あなたのECサイトにワンタグで埋め込める、TikTok風のショッパブル動画フィードです。サイト訪問者は動画を見ながらそのまま商品を購入できます。

あなた専用のタグを確認

以下があなたのサイト専用のウィジェットタグです。このタグ1つで全ての機能が動作します。

<!-- AitherHub Widget -->
<script src="https://www.aitherhub.com/widget/loader.js"
data-client-id="cab30223" async></script>
2

埋め込み方法を選択

お好みの方法でタグをサイトに追加してください。

GTM（推奨）簡単
HTML直接埋め込み
Google Tag Manager を使えば、サイトのコードを一切変更せずにウィジェットを追加・削除できます。エンジニアの作業は不要です。
  1. Google Tag Manager にログインします
  2. 対象のコンテナを選択し、「タグ」→「新規」をクリック
  3. タグの設定で「カスタム HTML」を選択
  4. 上記のタグ（Step 1）をそのまま貼り付け
  5. トリガーで「All Pages」を選択
  6. 「保存」→ 右上の「公開」ボタンを押す
GTMがまだサイトに導入されていない場合は、先にGTMのコンテナスニペットをサイトに追加する必要があります。GTM導入ガイド

サイトのHTMLテンプレートに直接タグを追加する方法です。

  1. サイトの共通テンプレート（header.php, layout.twig 等）を開く
  2. </body> タグの直前に、Step 1のタグを貼り付け
  3. ファイルを保存してデプロイ
<!-- サイトのHTML -->
<body>
... サイトのコンテンツ ...

<!-- AitherHub Widget（</body>の直前に追加） -->
<script src="https://www.aitherhub.com/widget/loader.js"
data-client-id="cab30223" async></script>
</body>
3

動作確認

タグを追加したら、サイトにアクセスして以下を確認してください。

  1. ページ右下にピンク色の丸いボタン（バブル）が表示される
  2. バブルをクリックするとフルスクリーンの動画フィードが開く
  3. 動画が自動再生され、上下スワイプで次の動画に切り替わる
  4. 「購入する」ボタンが表示され、クリックで商品ページに遷移する
GTM経由の場合、「プレビューモード」で公開前にテストすることもできます。

よくある質問

サイトの表示速度に影響はありますか？

ありません。ウィジェットは async 属性で非同期読み込みされるため、サイトの表示を一切ブロックしません。

動画の追加・変更はどうすればいいですか？

AitherHub管理画面からクリップを追加・削除できます。変更は即座にウィジェットに反映されます。サイト側の作業は不要です。

ウィジェットのデザインをカスタマイズできますか？

テーマカラー、CTAボタンの文言、ブランド名はAitherHub側で設定可能です。お客様のブランドに合わせた表示になります。

ウィジェットを一時的に非表示にしたい場合は？

GTM経由の場合はタグを「一時停止」にするだけです。HTML直接の場合はタグを削除またはコメントアウトしてください。

サポートが必要ですか？

セットアップで困ったことがあれば、お気軽にお問い合わせください。
